# Hangupphone (挂断垃圾电话)

> **项目简介**：一款基于 Android 原生 API 开发的纯净、无广告、完全本地运行的骚扰电话及陌生来电自动挂断拦截工具。

⚠️ **关于鸿蒙 NEXT (HarmonyOS NEXT) 的说明**：
* 本项目是基于标准 Android SDK（Kotlin/Room）开发的。
* **支持范围**：支持标准 Android 10+ (API 29+) 系统，以及具备安卓应用兼容运行环境的旧版鸿蒙系统（如 HarmonyOS 4.0 及以下版本，通过设置为默认来电筛选应用生效）。
* **鸿蒙 NEXT 兼容性**：由于 **HarmonyOS NEXT（纯血鸿蒙）** 已完全移除安卓应用兼容层，本项目**无法**在 HarmonyOS NEXT 系统上运行。

---

## ✨ 核心功能

1. **自动挂断/拦截**：接收到来电时，本软件会自动在后台进行过滤，如判定为骚扰电话，将直接向系统发送挂断指令，实现零打扰。
2. **通讯录白名单过滤**：支持“白名单防护模式”，仅允许保存在手机通讯录中的联系人拨入，非通讯录内的所有陌生号码一律拦截。
3. **本地黑名单管理**：支持手动将特定电话号码加入黑名单，或批量导入号码，实现精准定点拦截。
4. **拦截历史日志查看**：提供详细的拦截日志记录列表，展示拦截号码、拦截时间、拦截原因等。
5. **误杀一键恢复**：若发生误杀，可在拦截日志中一键选择“误杀恢复”，该号码将自动从黑名单中移出，保证后续正常呼入。
6. **运行状态诊断与事件记录**：界面自带系统回调诊断日志，方便排查“拦截服务未生效/被系统清理”等疑难杂症。
7. **轻量本地 AI 辅助打标（预留）**：已拦截的通话可异步调用轻量级分类器打标（预留 ONNX/TFLite 本地推理接口），全程无需联网。

## 🛠 开发与编译环境

为方便其他开发者快速配置环境并参与开发，以下为本项目推荐的开发环境参数：

* **开发工具 (IDE)**：Android Studio Giraffe (2022.3.1) 或更高版本
* **JDK 版本**：JDK 17
* **构建工具 (Gradle)**：Gradle 8.5
* **Android Gradle 插件 (AGP)**：8.5.0
* **核心编译参数**：
  * `compileSdk` = 34 (Android 14)
  * `targetSdk` = 30 (确保系统行为符合 `CallScreeningService` 稳定兼容规范)
  * `minSdk` = 29 (Android 10.0，因系统 `RoleManager.ROLE_CALL_SCREENING` 自该版本引入)
  * `Room Database` = 2.6.1

## 🚀 编译、安装与使用指南

由于本项目完全开源且未上架任何应用商店，您需要通过源码自行编译安装：

### 1. 开启手机开发者模式
1. 打开手机的 **设置 -> 关于手机**。
2. 连续点击 **版本号** 5~7 次，直至系统提示“您已处于开发者模式”。
3. 返回上一级设置，找到并进入 **开发者选项**。
4. 开启 **USB 调试**。

### 2. 编译与真机运行
1. 使用 USB 数据线连接手机与电脑，并在手机上弹出的提示中允许“USB 调试”。
2. 使用 Android Studio 打开本项目的根目录文件夹。
3. 等待 Gradle 依赖加载与索引构建完成。
4. 在 IDE 顶部设备选择栏中，选中您的真机设备。
5. 点击 **Run 'app'** 按钮（或按下 `Shift + F10`），Android Studio 将会自动完成编译、打包并安装到您的手机上。

> **关于签名**：在调试模式（Debug）下，Android Studio 会自动使用本地默认的 `debug.keystore` 进行签名，无需手动配置签名证书即可直接安装。若要生成发布版 APK，请在 Android Studio 中通过 `Build -> Generate Signed Bundle / APK` 生成您的专属签名密钥并进行签名打包。

### 3. 必须配置的系统设置（重要）
由于 Android 系统的安全限制，防骚扰软件安装后**必须**手动配置以下两项，拦截功能才能生效：

* **设置默认来电筛选应用（必须）**：
  * 打开本 App，按照“设置向导”提示，授予必要的权限。
  * 点击 **“来电筛选服务（必须）”** 按钮，在系统弹出的对话框中，将 **HangupPhone** 设置为 **“默认来电筛选应用”**（或在手机系统设置 -> 应用 -> 默认应用 -> 来电筛选/电话助手中手动设置）。
* **华为/荣耀机型后台保护配置（可选但强烈推荐）**：
  * 华为鸿蒙系统极易在后台自动清理服务。
  * 请进入手机的 **设置 -> 电池 -> 应用启动管理**。
  * 找到 **HangupPhone**，将“自动管理”切换为 **“手动管理”**。
  * 在弹出的选项中，勾选 **“允许自启动”**、**“允许关联启动”** 以及 **“允许后台活动”**。

### 4. 📲 首次使用权限配置向导与华为机型适配说明

当您在手机上成功安装应用后，首次启动将进入 **“权限设置向导”** 界面。请按以下 4 步依次操作：

| 步骤 | 界面操作按钮 | 作用描述 |
| :---: | :--- | :--- |
| **1** | **授予通讯录权限** | 白名单过滤基础：只放行您手机通讯录内的联系人来电。 |
| **2** | **授予电话状态权限** | 用于在有来电时实时识别来电号码。 |
| **3** | **授予来电筛选（核心）** | 设为默认来电筛选应用，使 `CallScreeningService` 能够执行后台挂断。 |
| **4** | **设为默认拨号（可选）** | 备选挂断方案，华为部分机型在无法回调来电筛选服务时需要。 |

完成以上设置后，点击 **“进入应用”** 即可。主界面中亦提供 **“重新打开权限设置向导”** 按钮以供随时调整。

#### 📱 华为 / 荣耀机型（以华为手机 / 鸿蒙系统为例）手动授权常见故障排除

在部分华为或荣耀的定制 ROM 上，由于系统权限管控极严，直接通过应用内点击可能会“没反应”或“找不到默认来电筛选选项”。此时，您需要按照以下路径前往系统设置中进行手动授权：

* **手动授予「来电筛选（核心，必须）」角色**：
  * 路径：**设置** → **应用** → **默认应用**。
  * 找到 **“来电筛选”** 或 **“来电助手”** / **“Caller ID & spam”** 选项（名称因系统版本而异）。
  * 将其默认应用选择并设为 **HangupPhone**。
* **手动授予基础权限（通讯录及电话状态）**：
  * 路径：**设置** → **应用** → **应用管理** → 找到并点击 **HangupPhone** → **权限**。
  * 手动允许 **“通讯录 / 联系人”** 以及 **“电话 / 电话状态”** 权限。
* **系统层手动备用路径（应用打不开时的备选方案）**：
  * 直接进入系统 **设置** → **应用** → **默认应用**，把来电筛选设为 **HangupPhone**。
  * 并且在 **设置** → **应用** → **HangupPhone** → **权限** 中，开启“通讯录/联系人”和“电话/电话状态”即可。

> 💡 **核心原理提示**：如果没有在系统内将本应用设为默认「来电筛选」应用，系统的 `CallScreeningService` 绝对不会被回调，拦截动作将无法生效。传统的 `BroadcastReceiver` 静态广播加电话管理反射手段在现代 Android / 鸿蒙系统上已无法可靠工作，必须使用该套最新的来电筛选接口。

## 🔒 敏感权限声明

作为一款防骚扰及来电拦截应用，本软件在运行中不可避免地需要向系统申请以下敏感权限。我们在下方公开了各权限的实际用途：

1.  **`android.permission.READ_CONTACTS` (读取联系人)**
    *   **用途**：用于查询来电号码是否属于您手机通讯录中的联系人，这是实现“通讯录白名单放行，非通讯录号码拦截”的核心前提。
2.  **`android.permission.READ_PHONE_STATE` (读取电话状态)**
    *   **用途**：用于获取和识别来电的实时状态与号码信息，用作辅助判定。
3.  **`android.permission.CALL_PHONE` 和 `android.permission.ANSWER_PHONE_CALLS` (拨打电话/接听电话)**
    *   **用途**：在特定机型或系统服务异常时，作为备选挂断方案（如通过 [FallbackCallEnder](file:///f:/MyProject/Hangupphone/app/src/main/java/com/hangupphone/service/FallbackCallEnder.kt)）挂断骚扰电话。
4.  **`android.permission.POST_NOTIFICATIONS` (发送通知)**
    *   **用途**：用于在拦截骚扰电话后，通过系统通知栏向您发送拦截详情通知，避免您漏掉重要信息。
